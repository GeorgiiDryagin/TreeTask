
# Инструкция по развертыванию (Deployment Guide)

У вас есть два пути: простой (Netlify/Vercel) и продвинутый (свой VPS).

## Вариант 1: Ваш VPS (Ubuntu/Debian)

Этот метод позволит запустить приложение на вашем сервере.

### 1. Подготовка сервера
Зайдите на сервер через SSH и установите Node.js (если еще нет):

```bash
# Обновляем пакеты
sudo apt update

# Устанавливаем Node.js (версия 20)
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# Проверяем установку
node -v
npm -v
```

### 2. Загрузка кода
Создайте папку и загрузите туда файлы проекта (можно через Git или FTP/SCP).

```bash
mkdir -p /var/www/treetask
cd /var/www/treetask
# (Здесь должны лежать файлы package.json, vite.config.ts и папка src/components...)
```

### 3. Установка зависимостей и Сборка
```bash
# Устанавливаем библиотеки
npm install

# Собираем проект (превращаем код в готовый сайт в папке /dist)
npm run build
```

### 4. Запуск (Простой способ)
Для теста можно запустить встроенный сервер предварительного просмотра. Он будет работать, пока открыт терминал.

```bash
# Запуск на порту 8080 (или любом другом)
npm run preview -- --host --port 8080
```
Теперь откройте в браузере: `http://IP-ВАШЕГО-VPS:8080`

### 5. Запуск (Постоянный способ - Nginx)
Чтобы сайт работал всегда и открывался по стандартному порту 80 (без :8080).

1. Установите Nginx: `sudo apt install nginx`
2. Скопируйте файлы из папки `dist` (которая появилась после build) в папку веб-сервера:
```bash
sudo cp -r dist/* /var/www/html/
```
3. Теперь сайт доступен просто по IP-адресу вашего сервера!

## Вариант 2: Netlify / Vercel (Рекомендую для старта)

Если вы не хотите настраивать Linux, это самый быстрый способ получить HTTPS ссылку.

1. Загрузите код на **GitHub**.
2. Зарегистрируйтесь на **Netlify** или **Vercel**.
3. Нажмите "New Project" -> "Import from GitHub".
4. Выберите репозиторий.
5. Настройки сборки определятся автоматически (Vite).
6. Нажмите **Deploy**.

Через 1 минуту вы получите ссылку вида `https://treetask-app.netlify.app`, которую можно открыть на телефоне.
